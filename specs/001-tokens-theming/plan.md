## Summary

Ship the Kimen theming system as pure compiled CSS: onmars as the default
theme (`:root`, light + dark via `prefers-color-scheme` with a
`data-ki-color-scheme` override) and material3 as the reference second theme
(`tokens.material3.css`, activated by `data-ki-theme='material3'`), built
exclusively by reassigning the theme and semantic DTCG layers over shared
primitives. Contract equality between themes, contrast floors and size
budgets are enforced by deterministic gates; scenarios S1-S7 are asserted in
the real-browser suite. onmars sources are already committed (generated from
MarsUI); material3 values come from the Material 3 Design Kit baseline
(research.md D2/D3).

**Scale/Scope**: ~854 custom properties per theme × 2 schemes; 2 themes; 7 scenarios (S1-S7)
